Get Started with Arduino & p5.js

This page will walk you through the basic moduals of FLEXIDOTS, and help you create your first wearable hardware experience!


Circuit Basics

For the first example, you will build a basic hardware system including a Microcontroller, one Input, and one Output.

Connect the Button to digital pin 2, and the Vibration Motor to digital pin 4. We are also using the on-board LED, which does not require an external module to be connected.
