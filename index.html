Hello World!

This is a site for the documentation of project Flexidots.

FlexiDots is a modular electronic kit specifically designed for wearable prototyping scenarios, which is addressing the challenge of making ergonomic, simplistic, and durable wearable electronics. This solution contains adaptive body attachments, optimized-wiring microcontroller & sensors kit, and software packages.
